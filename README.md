# 🔖 Bookmark Manager Website

A simple and clean **Bookmark Manager Website** built using **HTML** as part of my learning journey in the Sigma Web Development Course by CodeWithHarry.

---

## 🚀 Features

* 📌 Organized bookmarks into categories:

  * Primary Bookmarks
  * Secondary Bookmarks
* 🌐 Quick access to popular websites like:

  * Google
  * Instagram
  * YouTube
  * JioSaavn
  * Netflix
  * Amazon
  * Microsoft
* 🔗 Opens links in a new tab using `target="_blank"`
* 💡 Beginner-friendly project

---

## 🛠️ Tech Stack

* HTML5
